Elixir API wrapper for [CoinGecko](https://www.coingecko.com/) to fetch crypto
currencies prices.

## Installation

Add `coingecko` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:coingecko, "~> 0.1.0"}
  ]
end
```
